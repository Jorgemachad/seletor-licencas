# Seletor de Licenças

Um site para ajudar a selecionar licenças.


## Como instalar

Precisa de Python, Virtualenv e Ruby instalados.

    virtualenv env
    . env/bin/activate
    pip install -r requirements.txt
    
    gem install sass
    
## Como rodar para ele auto compilar

Precisa de Entr.
    
    ./run.sh
